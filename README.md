# Cliente Tarjetas de Video

Cliente Angular para la API de tarjetas de video. Permite consultar, crear, editar y eliminar tarjetas NVIDIA y AMD con un diseño oscuro.

## Tecnologías

- Angular 22
- TypeScript
- CSS (sin frameworks externos)

## Características

- Grid de tarjetas con filtro por marca (NVIDIA / AMD)
- Vista de detalle por tarjeta
- Formulario para crear y editar tarjetas
- Eliminación con confirmación
- Chips de color por marca

## Requisitos

Tener la [API Tarjetas Video](https://github.com/abomdev/api-tarjetas-video) corriendo localmente en `http://localhost:5070`.
